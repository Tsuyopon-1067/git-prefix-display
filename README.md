# git-prefix-display

git-prefix-displayはGitのコミットメッセージのプレフィックス候補を表示します．ただそれだけです．

# Usage
インストール後
```
gitpr
```
コマンドを実行．オプションはありません．

# Install
上部にある緑色のCodeボタンをクリックし，Download ZIPをクリックしてコードをダウンロード．解凍後そのディレクトリ内で
```
make install
```
コマンドを実行．
（もちろん普通にクローンしても問題なし）
